# 🚀 Dynamic File Compression Utility (Huffman Coding)

## 📌 Overview
This project is a **Huffman Coding based File Compression System** built using Python and Streamlit. It visually demonstrates how data compression works using greedy algorithms, binary trees, and priority queues.

---

## 🧠 DSA Concepts Used
- Huffman Tree (Binary Tree)
- Min Heap (Priority Queue)
- HashMap (Dictionary for frequency)
- Greedy Algorithm
- Recursion

---

## ⚙️ Features
- Upload text file
- Generate Huffman Codes
- Compress file into binary stream
- Decompress back to original text
- Visual Huffman Tree
- Compression statistics dashboard
- Interactive web UI

---

## 📊 Workflow
Input File → Frequency Table → Min Heap → Huffman Tree → Encoding → Compression → Decompression → Output
